# Single-Page CV

A simple and semantic single-page CV (Curriculum Vitae) built using only HTML. This project demonstrates the use of semantic HTML elements to create a clean and well-structured resume webpage.

## 📌 Project Overview

The goal of this project is to:

- Practice semantic HTML
- Create a structured CV layout
- Improve HTML fundamentals
- Build a responsive foundation for future CSS styling

## 🚀 Features

- Semantic HTML5 structure
- Single-page layout
- Personal information section
- Career objective
- Education section
- Skills section
- Experience section
- Projects section
- Contact information

## 🛠️ Technologies Used

- HTML5

## 📂 Project Structure

```
Single-Page-CV/
│
├── index.html
└── README.md
```

## ▶️ How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Double-click `index.html`
   **OR**
4. Open `index.html` in your preferred web browser.
